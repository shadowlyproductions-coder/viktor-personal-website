# Viktor Stojanoski Personal Website

Modern React, Tailwind CSS, Framer Motion, and Lucide React landing page for Viktor Stojanoski.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite, usually `http://127.0.0.1:5173`.

## Update placeholders

- Contact details: edit `src/data/siteData.js`.
- Profile visual: replace the placeholder card in `src/components/ProfileVisual.jsx` or add an image in `public/images/`.
- LinkedIn and CV buttons are currently removed from the live contact section.
